# Backend

## Overview
Automated pipeline that pulls live data, runs the FCI* estimation, and writes structured outputs consumed by the frontend.

## Folder structure
- `refs/matlab/` — read-only reference Matlab code (version 26_01_10); do not modify
- `data/raw/` — raw downloaded data; never hand-edited, always regenerated by the pipeline
- `data/output/` — estimation outputs consumed by the frontend (the interface contract)
- `pipeline/` — Python scripts for data ingestion, estimation, and output writing

## Output format (interface contract)

**`data/output/fcistar.csv`** — one row per quarter:
```
date, fci, fcistar, fci_gap, fcistar_upper, fcistar_lower
```
Dates in YYYY-MM-DD format. Confidence bands are for FCI*.

**`data/output/metadata.json`**:
```json
{
  "last_updated": "YYYY-MM-DD",
  "sample_start": "YYYY-MM-DD",
  "sample_end": "YYYY-MM-DD"
}
```

Any change to this format must be coordinated with the frontend.
